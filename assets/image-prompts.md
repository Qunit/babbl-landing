# Babbl — Website Image Prompts (copy-paste ready)

Each prompt below is **complete and standalone** — copy one, paste it into your image
generator, done. The device wording is identical in every device shot (only the
fabric colour changes) so the product looks consistent across the set.

**Device form:** a small **round, fabric-wrapped puck** — a smooth, gently domed disc
that sits low on the table (UFO/pebble-like, not a box), with a warm-white light ring
around the top edge and a single button on top. Same silhouette in every shot; only
the fabric colour changes.

**Fabric palette (chosen for the target group — calm, homely, never techy):** warm
oatmeal · warm sand · soft sage green · dusty terracotta (brand accent, product shots)
· muted grey-blue.

**Consistency tip:** generate `13` (macro) and `02` (still-life) first, pick the best
device render, then use it as an **image reference** for the rest — text alone drifts.
And the ring is **warm white, never red** (red reads as a recording light).
